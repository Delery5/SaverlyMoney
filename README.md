# Saverly Money

Saverly Money helps users track spending, monitor account balances, and build better budgeting habits. 
We use Plaid to securely connect to financial institutions. 
We **never** move money or make transactions on a user's behalf.

### What data we access:
- Account balances (read-only)
- Transaction history (read-only)

### Why we access data:
To help users budget intelligently and view their financial health in one dashboard.
